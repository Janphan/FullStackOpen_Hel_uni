This workspace contains multiple small projects used for the Full Stack Open exercises.

Projects:

- Root: learning playground with `vite` and root `package.json`.
- `anecdotes_app`: standalone React/Vite app (frontend).
- `simple phonebook`: React/Vite frontend for the phonebook exercise.
- `simple phonebook/backend`: Node/Express backend for the phonebook.

Usage

- Install all dependencies from the root using `npm install` (npm >=7 required for workspaces).
- Run a project's dev server from its folder, or use `npm run dev --workspace=<name>` (or refer to workspace manager) to run from root.
